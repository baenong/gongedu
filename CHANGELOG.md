# Changelog

## v0.9.5

**교육과정 주관부서 추가**

교육과정마다 담당 부서를 지정하고 조회할 수 있게 했습니다.

- 교육과정 등록 폼에 주관부서 선택 콤보박스 추가 (교육담당은 본인 부서로 자동 고정, 총괄담당 이상은 자유 선택)
- 교육 상세정보에 주관부서 표시, 총괄담당 이상은 상세정보에서 주관부서를 직접 수정 가능
- 캘린더 탭 우측 교육목록에도 주관부서 컬럼 표시

**캘린더 탭 UI 개선**

- 우측 교육목록 헤더(마감일/교육명/주관부서)를 스크롤 시에도 고정되도록 변경
- 헤더 정렬 및 구분선 정리, 화면 폭이 좁아 교육목록이 답답해지는 구간(895px~1019px)에서는 주관부서 컬럼을 숨겨 공간 확보
- 캘린더/목록이 나란히 배치되는 구간 중 목록 폭이 부족했던 896px~1069px 구간에서 캘린더 셀 너비를 살짝 줄여 목록에 공간을 더 배분
- 교육카드의 이수완료/미이수 뱃지에 좌측 여백을 줘 카드 폭이 좁을 때 D-Day/마감됨 표시와 붙어 보이던 문제 개선

**로고 및 브랜딩 변경**

- 로고 이미지 교체, 브라우저 탭 제목을 "공에듀"로 변경
- 로그인 화면/상단 헤더의 로고 크기 조정

## v0.9.4

**관리자 대리 이수 등록 기능 추가**

부서담당자 등 관리자가 교육 상세정보 모달에서, 직원이 시스템 밖에서 직접 제출한 수료증(대면·서면 제출, 또는 시스템 도입 이전 제출분)을 대신 등록할 수 있게 했습니다.

- 미제출 상태인 직원 행에 "대신 등록" 버튼 표시 (본인 행은 제외)
- 팀계담당(같은 팀)/부서담당(같은 부서)/교육담당(본인 등록 교육)/총괄담당 이상(전체) 관할 범위 내에서만 가능하도록 서버에서 검증
- 이미 제출된 건은 대리 등록으로 덮어쓸 수 없음

**버그 수정**

- 팀계담당/부서담당에게 교육과정 삭제·수정 버튼이 잘못 노출되던 문제 수정 (실제 처리는 서버에서 막고 있어 보안 문제는 아니었음)
- 업로드 파일이 1MB를 초과할 때 원인을 알 수 없는 오류 대신 안내 메시지가 표시되도록 수정
- 부서·팀명이 길 때 `이수완료`/`미이수` 뱃지가 두 줄로 쪼개지던 문제 수정

## v0.9.3

**메인화면 캘린더 뷰 추가**

상단 필터 바에 "캘린더" / "교육목록" 탭을 추가해, 마감일 기준 월간 캘린더와 상세 카드 목록을 화면 전환으로 나눠 볼 수 있게 했습니다.
카드 목록만으로는 파악하기 어려웠던 마감일 분포를 캘린더로 한눈에 확인할 수 있습니다.

- 캘린더 탭: 좌측 캘린더 + 우측 간단 목록(상태색 점 + 교육명 (마감일))을 나란히 표시
- 교육목록 탭: 기존 상세 카드 그리드가 화면 전체 너비를 사용
- 날짜 칸에 마감 교육을 상태별(마감임박=빨강, 일반=주황, 이수완료=초록) 색상 박스로 표시하고, 화면 폭에 따라 표시 개수를 조정해 초과분은 "+N개 더보기"로 축약
- 캘린더의 날짜나 간단 목록의 교육명을 클릭하면 교육목록 탭으로 전환되면서 해당 카드로 스크롤 이동 후, 무지개색 빛이 테두리를 따라 흐르는 애니메이션으로 약 4초간 강조 표시

**엑셀 일괄 등록 양식 수정**

직원 엑셀 일괄 등록 양식이 실제 업로드 처리 로직과 열 개수가 맞지 않아 등록이 실패하던 문제를 수정했습니다.

**IP 화이트리스트 초기값 수정**

신규 설치 시 IP 대역 설정 초기값이 `127.0.0.1`로 들어가 외부 접속이 전부 차단되던 문제를 수정했습니다. 이제 초기값은 빈 값(전체 허용)입니다.

## v0.9.2

**Docker 배포 지원**

서버에 Node.js를 직접 설치하지 않아도 Docker만으로 실행할 수 있도록 배포 방식을 개선했습니다.
`setup.bat` (Windows) 또는 `setup.sh` (Linux/macOS) 실행 한 번으로 환경 설정부터 컨테이너 실행까지 자동으로 완료됩니다.

**SSH 터널을 통한 초기 설정 지원**

PuTTY 등 SSH 클라이언트로 서버에 원격 접속하는 환경에서도 `geadmin` 계정으로 로그인할 수 있도록 개선했습니다.
SSH 로컬 포트 포워딩을 설정하면 관리자 PC의 브라우저에서 서버 localhost로 접속할 수 있습니다.

## v0.9.1

**교육담당 ZIP 파일명 개선**

교육담당 권한으로 이수증을 ZIP으로 다운로드할 때, 파일명이 교육담당자 본인의 부서명으로 고정되던 문제를 수정했습니다.
이제 이수 현황 화면에서 설정한 필터에 따라 파일명이 자동으로 결정됩니다.

| 필터 상태               | ZIP 파일명 형식                |
| ----------------------- | ------------------------------ |
| 모든 부서 (필터 없음)   | `교육명_시간.zip`              |
| 특정 부서로 필터        | `[부서명]교육명_시간.zip`      |
| 특정 부서 + 팀으로 필터 | `[부서명]팀명_교육명_시간.zip` |
