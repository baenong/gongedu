# Changelog

## v0.9.3

**메인화면 캘린더 뷰 추가**

상단 필터 바에 "캘린더" / "교육목록" 탭을 추가해, 마감일 기준 월간 캘린더와 상세 카드 목록을 화면 전환으로 나눠 볼 수 있게 했습니다.
카드 목록만으로는 파악하기 어려웠던 마감일 분포를 캘린더로 한눈에 확인할 수 있습니다.

- 캘린더 탭: 좌측 캘린더 + 우측 간단 목록(상태색 점 + 교육명 (마감일))을 나란히 표시
- 교육목록 탭: 기존 상세 카드 그리드가 화면 전체 너비를 사용
- 날짜 칸에 마감 교육을 상태별(마감임박=빨강, 일반=주황, 이수완료=초록) 색상 박스로 표시하고, 화면 폭에 따라 표시 개수를 조정해 초과분은 "+N개 더보기"로 축약
- 캘린더의 날짜나 간단 목록의 교육명을 클릭하면 교육목록 탭으로 전환되면서 해당 카드로 스크롤 이동 후, 무지개색 빛이 테두리를 따라 흐르는 애니메이션으로 약 4초간 강조 표시

**엑셀 일괄 등록 양식 수정**

직원 엑셀 일괄 등록 양식이 실제 업로드 처리 로직과 열 개수가 맞지 않아 등록이 실패하던 문제를 수정했습니다.

**IP 화이트리스트 초기값 수정**

신규 설치 시 IP 대역 설정 초기값이 `127.0.0.1`로 들어가 외부 접속이 전부 차단되던 문제를 수정했습니다. 이제 초기값은 빈 값(전체 허용)입니다.

## v0.9.2

**Docker 배포 지원**

서버에 Node.js를 직접 설치하지 않아도 Docker만으로 실행할 수 있도록 배포 방식을 개선했습니다.
`setup.bat` (Windows) 또는 `setup.sh` (Linux/macOS) 실행 한 번으로 환경 설정부터 컨테이너 실행까지 자동으로 완료됩니다.

**SSH 터널을 통한 초기 설정 지원**

PuTTY 등 SSH 클라이언트로 서버에 원격 접속하는 환경에서도 `geadmin` 계정으로 로그인할 수 있도록 개선했습니다.
SSH 로컬 포트 포워딩을 설정하면 관리자 PC의 브라우저에서 서버 localhost로 접속할 수 있습니다.

## v0.9.1

**교육담당 ZIP 파일명 개선**

교육담당 권한으로 이수증을 ZIP으로 다운로드할 때, 파일명이 교육담당자 본인의 부서명으로 고정되던 문제를 수정했습니다.
이제 이수 현황 화면에서 설정한 필터에 따라 파일명이 자동으로 결정됩니다.

| 필터 상태               | ZIP 파일명 형식                |
| ----------------------- | ------------------------------ |
| 모든 부서 (필터 없음)   | `교육명_시간.zip`              |
| 특정 부서로 필터        | `[부서명]교육명_시간.zip`      |
| 특정 부서 + 팀으로 필터 | `[부서명]팀명_교육명_시간.zip` |
